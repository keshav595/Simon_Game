Memory Game

press any key to start the game.

    


Instructions

How to Play

  • The game will flash you a colour box.
  • You have to memorize the colour and repeat it by clicking on the color blocks .
  • For each correct , you will proceed to the next level with an additional color added.
  • You have to memorize the sequence and repeat it by clicking on the color blocks in the same order.
  • If you make a mistake, the game will end and your score will be displayed.
  • Try to get the highest score!
